Responsive Design

Responsive web design is an approach to web design that makes web pages render well on a variety of devices and window or screen sizes, from minimum to maximum display size.

1. Viewport Meta Tag

The `` tag is essential for responsive design. It tells the browser how to control the page's dimensions and scaling. 

<meta name="viewport" content="width=device-width, initial-scale=1.0">
  • `width=device-width`: Sets the width of the page to follow the screen-width of the device.
  • `initial-scale=1.0`: Sets the initial zoom level when the page is first loaded by the browser.

2. Media Queries (`@media`)

Media queries allow you to apply CSS styles based on device characteristics (e.g., width, height, orientation, resolution).

Try resizing your browser window to see the layout change:

Column 1 (Small: 100%, Medium: 50%, Large: 33.33%)
Column 2 (Small: 100%, Medium: 50%, Large: 33.33%)
Column 3 (Small: 100%, Medium: 50%, Large: 33.33%)
Column 4 (Small: 100%, Medium: 50%, Large: 33.33%) 
/* Styles for screens smaller than 600px (Mobile First) */
body { font-size: 16px; }

@media screen and (min-width: 600px) {
  /* Styles for screens 600px and wider */
  body { font-size: 18px; }
}

@media screen and (min-width: 900px) {
  /* Styles for screens 900px and wider */
  body { font-size: 20px; }
}

Common Media Features:

  • `width`, `min-width`, `max-width`
  • `height`, `min-height`, `max-height`
  • `orientation` (portrait or landscape)
  • `resolution`

3. Flexible Units

Using relative units instead of absolute units allows elements to scale proportionally with the viewport or parent elements.

Percentage (`%`)

Relative to the parent element's size.

This div is 80% of its parent's width. 
width: 80%;

`em` and `rem`

  • `em`: Relative to the `font-size` of the element's *parent*.
  • `rem`: Relative to the `font-size` of the *root* (html) element. Generally preferred for consistent scaling.

This text uses `rem` units.

This text uses `em` units (parent font-size is 20px). 

font-size: 1.2rem; /* relative to root */
font-size: 1.2em; /* relative to parent */

Viewport Units (`vw`, `vh`, `vmin`, `vmax`)

  • `vw`: 1% of the viewport's width.
  • `vh`: 1% of the viewport's height.
  • `vmin`: The smaller of `vw` or `vh`.
  • `vmax`: The larger of `vw` or `vh`.
This div is 80vw wide and 10vh tall.
width: 80vw;
height: 10vh;

4. Flexible Images

Images should scale down to fit their container without overflowing. 

img {
  max-width: 100%;
  height: auto;
  display: block;
}
