CSS Transforms

CSS transforms allow you to move, rotate, scale, and skew elements. They provide a powerful way to create visual effects and animations.

1. 2D Transforms

These transforms affect elements in a two-dimensional plane (X and Y axes).

`translate()`

Moves an element from its current position.

Translate
transform: translate(50px, 20px); /* Moves 50px right, 20px down */
transform: translateX(50px);
transform: translateY(20px);

`rotate()`

Rotates an element around its central point (by default).

Rotate
transform: rotate(45deg);

`transform-origin`

Allows you to change the position of the transformed element's origin (the point around which a transform is applied).

Rotate (Origin Top Left)
transform-origin: top left; /* Default is center center */

`scale()`

Increases or decreases the size of an element.

Scale
transform: scale(1.2); /* 1.2 times original size */
transform: scaleX(0.8);
transform: scaleY(1.5);

`skew()`

Skews an element along the X and Y axes.

Skew
transform: skew(20deg, 10deg); /* Skew X by 20deg, Y by 10deg */
transform: skewX(20deg);
transform: skewY(10deg);

`matrix()` (Advanced)

Combines all 2D transform methods into one. It takes six values.

Matrix
transform: matrix(scaleX, skewY, skewX, scaleY, translateX, translateY);

2. 3D Transforms

These transforms allow you to manipulate elements in three-dimensional space.

For 3D transforms to be visible, the parent element often needs a `perspective` property.

`rotateX()`

Rotates an element around its X-axis (horizontal axis).

Rotate X
transform: rotateX(180deg);

`rotateY()`

Rotates an element around its Y-axis (vertical axis).

Rotate Y
transform: rotateY(180deg);

`rotateZ()`

Rotates an element around its Z-axis (an axis perpendicular to the screen).

Rotate Z
transform: rotateZ(90deg);

`translateZ()`

Moves an element along the Z-axis (towards or away from the viewer).

Translate Z
transform: translateZ(100px);

`perspective` (on parent)

Defines the perspective from which the 3D transformed elements are viewed. 

.container {
  perspective: 800px;
}

`backface-visibility`

Determines whether the back face of an element is visible when turned towards the user.

Backface Hidden
backface-visibility: hidden;

Exercises

  1. Apply a `translate` transform to a `div` that moves it diagonally (e.g., `translate(30px, 30px)`).
  2. Create a `div` and apply `rotateY(180deg)` on hover. What happens to the text? Then, add `backface-visibility: hidden;` to see the effect.
  3. Combine `scale` and `rotate` transforms on a single element on hover.
  4. Create a 3D cube effect using multiple `div` elements and `rotateX`, `rotateY`, `translateZ` transforms. (Advanced)
