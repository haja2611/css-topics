Units & Values

CSS uses various units to specify lengths, sizes, and other values, as well as different formats for defining colors.

1. Length Units

Absolute Length Units (Fixed sizes, not recommended for responsive design)

  • `px` (Pixels): Relative to the viewing device. 1px = 1/96th of 1 inch.
  • `pt` (Points): 1pt = 1/72 of 1 inch. Primarily used in print media.
  • `cm`, `mm`, `in` (Centimeters, Millimeters, Inches): Physical units, also primarily for print.
150px Width

This text is 14pt.

width: 150px;
font-size: 14pt;

Relative Length Units (Scale relative to something else, great for responsiveness)

  • `em`: Relative to the font-size of the element itself (or its parent, if not set on element).
  • `rem`: Relative to the font-size of the root (``) element.
  • `%` (Percentage): Relative to the parent element's size.
  • `vw` (Viewport Width): 1% of the viewport width.
  • `vh` (Viewport Height): 1% of the viewport height.
  • `vmin`: 1% of the smaller of viewport width or height.
  • `vmax`: 1% of the larger of viewport width or height.

2. Color Values

CSS supports various ways to define colors:

  • Named Colors: `red`, `blue`, `green`, `dodgerblue`, etc.
  • Hexadecimal (HEX): `#RRGGBB` (e.g., `#FF0000` for red), `#RGB` (shorthand)
  • RGB/RGBA: `rgb(red, green, blue)` or `rgba(red, green, blue, alpha)` (alpha for opacity 0-1)
  • HSL/HSLA: `hsl(hue, saturation, lightness)` or `hsla(hue, saturation, lightness, alpha)`

3. Other Common Values

  • `auto`: Often used for margins (`margin: 0 auto;`) to center block elements, or for sizing where the browser calculates the value.
  • `none`: Used to remove a property value (e.g., `list-style: none;`, `text-decoration: none;`).
  • `initial`: Sets a property to its default value.
  • `inherit`: Inherits the property value from its parent element.

Exercises

  1. Create a new `div` with a `width` of `50vw` and a `font-size` of `3vh`. Observe how they change upon resizing the browser.
  2. Set the `margin-top` of a paragraph to `2rem` and its `padding-bottom` to `1em`. Explain the difference.
  3. Create a `div` and set its `background-color` using an HSLA value with `50%` opacity.
  4. Using the `initial` keyword, reset the `font-weight` of an `h1` element inside a specific `div` to its browser default.
