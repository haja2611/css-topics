Exercises

  1. Create a new div with a content width of 100px, 15px padding on all sides, a 3px dashed green border, and 25px margin on all sides. Use `box-sizing: content-box;`.
  2. Create another div with a total width of 200px, 10px padding, and a 5px solid blue border. Use `box-sizing: border-box;`. Compare their actual sizes using browser developer tools.
  3. Apply different padding values (e.g., top: 5px, right: 10px, bottom: 15px, left: 20px) to an existing element.
  4. Experiment with `margin-top` and `margin-bottom` on two paragraphs to see how vertical margins collapse.
