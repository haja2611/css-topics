Pseudo-classes & Elements

CSS pseudo-classes and pseudo-elements are used to define special states of an element or to style specific parts of an element, respectively.

Pseudo-classes

A pseudo-class is used to define a special state of an HTML element. It starts with a single colon (`:`).

1. `:hover`

Selects elements when you mouse over them.

Hover over me
.element:hover {
  background-color: blue;
}

2. `:active`

Selects the active link or element (e.g., when a button is clicked).

button:active {
  background-color: darkgreen;
}

3. `:focus`

Selects an element that has input focus (e.g., when a text field is clicked or tabbed into). 

input:focus {
  border-color: orange;
}

4. `:first-child`

Selects every element that is the first child of its parent.

This is the first paragraph (first-child).

This is the second paragraph.

This is a span. 
p:first-child {
  font-weight: bold;
}

5. `:nth-child(n)`

Selects every element that is the nth-child of its parent. `n` can be a number, keyword (odd, even), or a formula (an+b).

  • List Item 1 (odd)
  • List Item 2 (even)
  • List Item 3 (odd)
  • List Item 4 (even)
li:nth-child(even) {
  background-color: lightgray;
}
li:nth-child(2n+1) { /* Same as odd */
  color: blue;
}

6. `:not(selector)`

Selects every element that is NOT the specified selector.

  • Item 1
  • Special Item
  • Item 3
li:not(.special) {
  color: gray;
}

Pseudo-elements

A CSS pseudo-element is used to style a specified part of an element. It starts with a double colon (`::`).

1. `::before`

Inserts content before the content of an element.

This is a quoted text example.

.element::before {
  content: "\201C";
}

2. `::after`

Inserts content after the content of an element.

Visit Website 
.element::after {
  content: "\2192";
}

3. `::first-line`

Applies styles to the first line of a block-level element.

This is a long introductory paragraph to demonstrate the `::first-line` pseudo-element. As you can see, only the very first line of this text will be affected by the styles applied to this pseudo-element, while the rest of the text remains in its default styling. 

p::first-line {
  font-variant: small-caps;
}

4. `::first-letter`

Applies styles to the first letter of the first line of a block-level element.

This paragraph starts with a drop cap, where the first letter is significantly larger and styled differently from the rest of the text, creating a visually appealing effect often found in traditional typography. 

p::first-letter {
  font-size: 3em;
  float: left;
}

5. `::selection`

Applies styles to the portion of an element that is highlighted by the user (e.g., selected text).

Try selecting this text to see the `::selection` pseudo-element in action. 

::selection {
  background-color: yellow;
  color: black;
}
