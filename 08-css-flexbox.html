Exercises

  1. Create a flex container with 4 items. Set `justify-content` to `space-between` and `align-items` to `center`.
  2. In the same container, make the second item `flex-grow: 2` and the fourth item `flex-grow: 0.5`.
  3. Create a new flex container with `flex-direction: column;` and `align-items: flex-end;`.
  4. Experiment with `flex-wrap: wrap-reverse;` and `align-content` (for multi-line flex containers) to see how items align when wrapped.
