Advanced Selectors

Beyond basic type, class, and ID selectors, CSS offers more powerful and flexible selectors to target elements with greater precision.

1. Combinators

Combinators explain the relationship between the selectors.

Descendant Selector (Space)

Selects all elements that are descendants of a specified element (e.g., `div p`).

This is a paragraph inside the container.

This paragraph is also a descendant (inside a span).

 
.container p { color: pink; }

Child Selector (>)

Selects all elements that are direct children of a specified element (e.g., `ul > li`).

This is a direct child paragraph.

This paragraph is NOT a direct child.

 
.child-container > p { background-color: lightgreen; }

Adjacent Sibling Selector (+)

Selects an element that is directly placed after another specified element, at the same level.

Section A

This paragraph is an adjacent sibling to the heading above.

This paragraph is NOT an adjacent sibling.

h3 + p { margin-top: 20px; }

General Sibling Selector (~)

Selects all elements that are siblings of a specified element, regardless of their position.

Section B

This paragraph is a general sibling.

Another element.

This paragraph is also a general sibling.

h3 ~ p { text-decoration: underline; }

2. Advanced Attribute Selectors

These allow you to select elements based on specific patterns in their attributes.

Starts With ([attribute^=value])

Selects elements whose specified attribute value begins with a specified string. 

[class^="btn-"] { /* ... */ }

Ends With ([attribute$=value])

Selects elements whose specified attribute value ends with a specified string.

Download PDF

Download ZIP

Go to Home

[href$=".pdf"] { color: purple; }

Contains ([attribute*=value])

Selects elements whose specified attribute value contains a specified string.

Important Info 1
Regular Info
Important Info 2
[data-info*="important"] { border: 2px solid red; }

3. Structural Pseudo-classes

These pseudo-classes target elements based on their position within the document structure.

:nth-of-type(n)

Selects every element that is the `n`th child of a *particular type* among its siblings.

Heading 1

Paragraph 1

Heading 2

Paragraph 2

Span 1 Span 2 
p:nth-of-type(odd) { background-color: lightpurple; }
span:nth-of-type(even) { background-color: lightblue; }

:first-of-type

Selects the first sibling of its type.

:last-of-type

Selects the last sibling of its type.

:only-child

Selects an element that is the only child of its parent.

I am the only child.

I have siblings.

I have siblings.

p:only-child { background-color: yellow; }

:empty

Selects elements that have no children (including text nodes).

.empty-box:empty { background-color: lightgreen; height: 30px; }

Exercises

  1. Using the child selector (`>`), style only the direct `span` children of a `div` element to have a `font-weight: bold;`.
  2. Create a list of files (`.txt`, `.doc`, `.jpg`) and use attribute selectors to style each file type with a different color (e.g., green for `.txt`, blue for `.doc`, red for `.jpg`).
  3. Style every third `li` element in a `ul` using `:nth-child(3n)`.
  4. Create a `div` that contains only text. Apply a style to it using `:only-child` and then add another element inside to see the style removed.
