Text & Fonts

CSS offers a wide range of properties to control the appearance of text and fonts.

1. Font Family (`font-family`)

Specifies the font for an element. Always provide a fallback generic family.

This text is in a serif font (e.g., Times New Roman).

This text is in a sans-serif font (e.g., Arial).

This text is in a monospace font (e.g., Courier New). 

font-family: "Arial", sans-serif;

2. Font Size (`font-size`)

Sets the size of the font. Can use `px`, `em`, `rem`, `%`, etc.

This is small text (0.8em).

This is large text (1.5em).

This is 24px text.

font-size: 16px; /* Absolute size */
font-size: 1.2em; /* Relative to parent's font-size */

3. Font Weight (`font-weight`)

Sets the boldness of the font.

This text is light (300).

This text is bold.

This text is bolder.

font-weight: bold; /* Keywords */
font-weight: 700; /* Numeric values (100-900) */

4. Font Style (`font-style`)

Sets the font style to normal, italic, or oblique.

This text is italic.

font-style: italic;

5. Text Alignment (`text-align`)

Aligns the text within an element horizontally.

This text is centered.
This text is right-aligned.

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. 

text-align: center;

6. Line Height (`line-height`)

Sets the height of a line of text.

This text has a tight line height.
Notice the reduced space between lines.

This text has a loose line height.
Notice the increased space between lines. 

line-height: 1.6; /* Unitless value is relative to font-size */

7. Text Decoration (`text-decoration`)

Applies various text decorations.

This text is underlined.

This text has a line through it.

This link has no underline. 
text-decoration: underline;

8. Text Transform (`text-transform`)

Controls the capitalization of text.

This text is uppercase.

THIS TEXT IS LOWERCASE.

this text is capitalized.

text-transform: uppercase;

9. Letter Spacing (`letter-spacing`)

Increases or decreases the space between characters.

W I D E S P A C I N G

TightSpacing

letter-spacing: 2px;

10. Word Spacing (`word-spacing`)

Increases or decreases the space between words.

This text has wider word spacing.

word-spacing: 5px;

11. Text Shadow (`text-shadow`)

Adds shadow to text.

Text with Shadow

text-shadow: 2px 2px 4px #aaa;

Exercises

  1. Change the `font-serif` class to use a different serif font like "Georgia".
  2. Set the `line-height` of all paragraphs to `1.8` using a type selector.
  3. Create a new class `strike-through` that applies `text-decoration: line-through;` and apply it to a sentence.
  4. Apply a text shadow to all `h1` elements with a color of `gray` and a blur radius of `3px`.
