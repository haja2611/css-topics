Exercises

  1. Modify the `columns` layout: for screens wider than `768px`, make it a 3-column layout.
  2. Add a media query that changes the `background-color` of the `body` to `lightgreen` when the screen orientation is `portrait` and `max-width` is `480px`.
  3. Create a new `p` element. Set its `font-size` using `vw` units and observe how it scales with the browser window width.
  4. Implement a simple navigation bar that changes from a horizontal layout on large screens to a stacked vertical layout on small screens using a media query.
