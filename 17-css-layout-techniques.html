Layout Techniques (Float & Clear)

Before Flexbox and Grid, `float` and `clear` were commonly used for creating multi-column layouts and wrapping text around images. While still valid, modern CSS layout methods are generally preferred for complex layouts.

1. `float` Property

The `float` property is used to place an element on the left or right side of its container, allowing text and inline elements to wrap around it. It takes the element out of the normal document flow.

`float: left;`

3. Clearfix Techniques

When you float elements inside a container, the container might collapse (its height becomes zero) because floated elements are out of the normal document flow. Clearfix methods are used to fix this.

`overflow: auto;` or `overflow: hidden;`

Adding `overflow: auto;` or `overflow: hidden;` to the parent container can force it to contain its floats.

  overflow: auto; /* Or hidden */

`::after` Pseudo-element Clearfix (Recommended)

This method adds a generated content after the floated elements and clears it, forcing the container to expand its height.

Use Cases & Limitations

Float-based layouts were standard for a long time for:

  • Creating multi-column layouts.
  • Wrapping text around images.

Limitations: Not designed for complex layouts, often requires clearfix hacks, can be unpredictable with varying content, and not ideal for responsive design without extensive media queries.
